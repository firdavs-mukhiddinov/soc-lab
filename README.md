# Splunk SOC Lab

## Overview

This repository contains security monitoring and incident investigation projects completed in a home lab environment using Splunk Enterprise on Ubuntu.

## Objectives

- Practice SIEM operations using Splunk
- Develop detection queries
- Investigate security events
- Create incident reports
- Improve SOC analyst skills

## Use Cases

1. SSH Brute Force Detection
2. Privilege Escalation Attempt Detection
3. Failed Login Monitoring
4. Nmap Port Scan Detection
5. Root Login Monitoring

## Technologies Used

- Splunk Enterprise
- Ubuntu Linux
- SSH
- Linux Logs
- SPL (Search Processing Language)

## Skills Demonstrated

- Log Analysis
- Incident Investigation
- Alert Tuning
- Detection Engineering
- MITRE ATT&CK Mapping
- True Positive / False Positive Analysis
